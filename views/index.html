Timestamp App

Have a try.

Pass a time as a parameter to the url of this microservice.

This App will return a date json string with unix and natural dates for any valid query. Values of null will be returned for all non unix or natural date queries.


Example usage:

'https://timestamp-js.glitch.me/Oct%2013,%201989' or

'https://timestamp-js.glitch.me/October 13, 1989' or

'https://timestamp-js.glitch.me/624265200'

Example output:

{"unix":624265200,"natural":"October 13, 1989"}
